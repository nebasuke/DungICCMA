A command line application written on top of the Dung library satisfying the 
ICCMA guidelines.


------ Installation instructions ------
1. Install GHC 7.8.4 or newer (this should include cabal 1.18 needed for the cabal sandboxing) 
2. Run make or compile.bat dependent on OS to build the executable. 
3. The executable should appear as /dist/build/dungell_iccma/dungell_iccma.exe  or /dist/build/dungell_iccma/dungell_iccma (or alternatively in .cabal-sandbox\bin)
4. Run the executable using the probo standard command-line arguments.
